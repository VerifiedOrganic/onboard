## [Unreleased]

### Added

- MCP server support over stdio and Streamable HTTP.
- Tree-sitter-backed code graph tools for repo maps, trace flow, impact analysis, context packs, dead-code leads, and rendered maps.
- Structural scanners for dependencies, schemas, routes, stacks, and repository reconnaissance.
- Git-backed history, diff, churn, and explain-diff signals for change review.
- SHA-tagged guide cache for durable codebase walkthroughs.
- Agent installer, dry-run, doctor, and uninstall flows for supported MCP clients.
- Embedded onboarding, infrastructure, diagram, impact, risk, and guide-maintenance skills.
- First-class HCL/Terraform/Terragrunt/OpenTofu indexing and deadness rules.

This changelog starts at commit `00d68d5`; earlier project history is intentionally not reconstructed.
